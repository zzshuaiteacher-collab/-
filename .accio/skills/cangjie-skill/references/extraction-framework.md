# 仓颉 v2.0 提炼方法论：理-文-情-神 四重耦合

## 一、 理之提炼：心智模型的三重验证
每一个被收录的「模型」必须满足：
1. **跨域复现**：在不同类型的文章中反复出现其分析逻辑。
2. **生成力**：可以用此模型推断该团队对新热点的态度。
3. **排他性**：具有该团队独特的智力标签（如：远川的「底座」、42章经的「元问题」）。

**如果一个观点只通过了 1 重验证** → 降级为「选题偏好」而非「心智模型」。
**如果一个观点 0 重验证都没过** → 丢弃，可能只是某篇文章的即兴发挥。

## 二、 文之提炼：表达 DNA 的量化
通过对海量语料（含微信白名单）的扫描，提取以下指纹：

### 1. 词汇指纹
- **高频术语**：该行业中该团队偏好的特定词汇。
- **语气助词**：结尾习惯（嘛、呢、吧）或转折习惯（其实、毕竟、有趣的是）。
- **功能词频率**：统计「的」「了」「是」「但」「而」「其实」等高频功能词的使用密度。

### 2. 节奏指纹
- **平均句长**：精确到字数（如：28字/句）。
- **段落密度**：平均每段行数。
- **反问频率**：设问句占比（精确到百分比）。
- **感叹号密度**：每100字中感叹号数量。

### 3. 叙事模式
- **结论先行型**：开篇炸场，中间堆料，结尾升华。
- **剥洋葱型**：从细节切入，层层递进，最终揭示本质。
- **商业戏剧型**：强人设、强冲突、强对话，通过人物驱动逻辑。
- **冷峻白描型**：去修饰、去煽情，用精准的细节本身说话。

### 4. 排版指纹
- **标题格式**：数字编号（01/02/03）还是文字标题？
- **加粗规则**：每隔几段出现一次？加粗的是结论还是概念？
- **引号风格**：「直角引号」还是"弯引号"？
- **Emoji 使用**：频率和类型。
- **署名格式**：「作者｜XX」还是「文/XX」？

## 三、 情之提炼：情感基调的四维建模 ⭐v2.0 核心新增

### 维度 1: Valence (效价)
- 正面词/负面词/中性词在文本中的基准比例。
- 示例：晚点 = 正20% 负30% 中50%；新智元 = 正60% 负10% 中30%。

### 维度 2: Arousal (唤醒度)
- 高能/平静/压抑的段落占比。
- 示例：新智元 = 高能70%；42章经 = 平静80%。

### 维度 3: Empathy (共情模式)
| 模式 | 定义 | 典型代表 |
|------|------|---------|
| **俯视型** | 「我来告诉你真相」 | 晚点、远川 |
| **平视型** | 「我们一起来看看」 | 极客公园、42章经 |
| **仰视型** | 「这太牛了！」 | 新智元、量子位 |

### 维度 4: Tension Arc (情感弧线)
- **冷启动→逻辑递进→洞察高潮→冷收尾**（42章经）
- **震撼开场→参数堆叠→全面评估→展望未来**（新智元）
- **宏大开场→人物白描→逻辑拆解→哲思收尾**（晚点）
- **变量发现→人物叙事→认知升维→使命召唤**（极客公园）

### 情感温度计
对该团队的整体情感温度打分：
- 0 = 极度冷峻（如：晚点 ≈ 25）
- 50 = 中性平衡
- 100 = 极度热情（如：新智元 ≈ 85）

### 情感禁区
明确列出该团队 **绝对不会使用** 的情感表达方式：
- 示例（晚点）：禁止使用「震撼！」「炸场！」「太牛了！」
- 示例（新智元）：禁止使用「冷峻地说」「值得深思」等低能量表达

## 四、 神之进化：Correction 2.0 分层机制
当用户提出「不像」或「不对」时，执行以下操作：

### Step 1: 分类识别
根据反馈内容自动判断属于哪一层：
- `[C-理]`：逻辑层（如「他们不会用这个论证结构」）
- `[C-文]`：表达层（如「这里应该用短句而不是长句」）
- `[C-情]`：情感层（如「这段应该更冷峻/更兴奋」）
- `[C-排]`：排版层（如「加粗太多了」）

### Step 2: 生成补丁
在 `Correction.md` 中写入一条带有时间戳和分类标签的修正指令。

### Step 3: 权重复写
在下一次生成前，强制将补丁内容按以下优先级注入：
```
Correction.md [C-理/C-文/C-情/C-排] → 最高权重
Golden_Corpus.md → 第二权重
Writing_Pattern.json → 第三权重
SKILL.md 基础框架 → 基础权重
```

## 五、 微信白名单检索策略
1. **核心策略**：微信公众号是仓颉的一级水源，享有最高搜索权重。
2. **爆款采样**：优先采样过去 3 个月的 Top 5 爆款文章（按阅读量/点赞量排序）。
3. **定义之作采样**：采集该账号历史上的「里程碑文章」（如成名作、争议性文章）。
4. **视觉抓取**：记录文中加粗、倾斜、Emoji 的具体位置和频率。
5. **情感采样**：专门标注语料中的情感高峰和情感低谷段落。
