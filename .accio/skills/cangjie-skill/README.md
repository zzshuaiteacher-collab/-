# 仓颉.skill (Cangjie)
> *「不仅看透灵魂，还要穿上皮囊」*

**仓颉是「女娲」与「同事」的完美融合。它负责蒸馏任何团队、公众号或机构的写作风格，并将其转化为可运行的高仿生写作引擎。**

---

## 🌪️ 核心优势
1. **地毯式搜索**：8 路 Sub-Agents 并行，将微信公众号列为核心白名单，深度扫描历史语料。
2. **三位一体提炼**：
   - **理**：心智模型三重验证（女娲逻辑）。
   - **文**：写作指纹量化还原（同事风格）。
   - **神**：Correction 机制动态进化（用户纠错）。
3. **长文深度填充**：专门针对 3000 字以上深度内容优化，拒绝平庸的 AI 味。
